Criação de Listas

Tag < li > = item da Lista.

Tag < ul > para criar listas não ordenadas, utiliza apenas marcador.

  • Item 01

Tag < ol > para criar listas ordenadas, utilizando números para ordenar.

  1. Item 01

  3. Item 03

Criação de Tabelas

Tag < table > para criação de Tabelas.

Tag < thead > = Cabeçalho da Tabela.

Tag < tbody > = Corpo da Tabela.

Tag < tr > = TABLE ROW = Linha da Tabela.

Tag < th > = TABLE HEAD = Coluna da Tabela (em negrito).

Tag < td > = TABLE DATA = Conteúdo da Tabela (sem negrito)

Mesclar Colunas = < colspan > e entre aspas, a quantidades de colunas a se mesclar.

Mesclar Linhas = < rowspan > e entre aspas, a quantidade de linhas a se mesclar

Criação de Formulários

Tag < form > para criação de Formulários.

Tag < label for > define o rótulo (nome) do campo no formulário.

Tag < imput type> define o Tipo de conteúdo será aplicado ao respectivo campo no formulário.

Tag < required > torma o campo do formulário ao modo de preenchimento obrigatório.
